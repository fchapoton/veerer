How to install PPL:
(On Ubuntu this would just be: sudo apt-get install cython libgmp-dev libppl-dev)

Install gmpxx:
	Download gmp from https://gmplib.org/
	Run:
		./configure --enable-cxx
		make
		make check
		make install

Install PPL:
	Download ppl_c from http://bugseng.com/products/ppl/download
	Run:
		./configure
		make
		make install
